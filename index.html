🚀 Progress 2 · Direct DeepSeek API & 5‑Minute Agent Runtime

As of May 5, 2026, we have achieved:

Bypassed MCP – Direct DeepSeek API Calls ✅ Success
The original MCP (Model Context Protocol) tool caused persistent environment issues (port conflicts, symbol mismatches, LangChain version glue). We completely removed the MCP dependency and implemented a native Python function _call_deepseek_api() that directly sends price data to DeepSeek’s API and receives structured trading decisions.
Full 5‑Minute Agent Integration ✅ Done
Built BaseAgentAStock_5min_DeepSeek class that:
  • Reads merged_5min.jsonl → extracts 144 valid 5‑minute K‑lines
  • Filters decision points (default every 6 bars → 24 trading opportunities)
  • Retrieves OHLCV prices for each time point using local JSONL parsing (no MCP)
  • Formats a market snapshot and sends it to DeepSeek API with a system prompt requesting JSON decision output
  • Logs AI responses and updates a demonstration position file
Runtime Validation & Results ✅ Verified
Successfully processed all 24 decision points across three trading days (2026-04-28 to 2026-04-30). For each point, the Agent:
  • Retrieved real 5‑minute open/high/low/close/volume for 10 A‑share stocks
  • Called DeepSeek API (model deepseek-chat) with a concise prompt
  • Received and logged structured JSON decisions (e.g., {"decision":"hold","symbol":"600519.SH","price":1402.0,"reason":"..."})
No MCP services were required – the Agent runs standalone with only Python requests and the DeepSeek API key.
Code Artifact & Reproducibility
The modified Agent class is saved as agent/base_agent_astock/base_agent_astock_5min_deepseek.py. Configuration file configs/astock_5min_deepseek_config.json is ready. Running python main.py configs/astock_5min_deepseek_config.json reproduces the results.

📰 News Expert – Unified Market Intelligence

🎯 Goals

Reduce token consumption & decision rounds by providing a consolidated market digest (macro + sector + major stocks) to all agents.

⚙️ Architecture

Periodic retrieval of global news → LLM summarization → structured report (market sentiment, hot sectors, affected stocks). Agents receive this report as part of their system prompt; on‑demand search tool allows deeper dive.

✅ Benefits

– Covers multi‑level information (macro, industry, individual)
– Avoids redundant per‑stock news queries
– Enables agents to react to breaking events within 5‑minute windows
